1.What is Digital Certificate ?
An Digital-certificate that is issued electronically rather than in physical form. It is a digital document that serves as proof of completion or achievement in a particular course, program, or activity. E-certificates are becoming increasingly popular in today's digital age because they are more convenient and efficient than traditional paper certificates. They can be easily shared and stored electronically, and they are accessible from anywhere with an internet connection.

2.What is Block-Chain ?
A blockchain is a distributed and decentralized digital ledger that records transactions in a secure and tamper-resistant manner. It consists of a continuously growing list of records called blocks, which are linked and secured using cryptography.
Each block typically contains a cryptographic hash of the previous block, a timestamp, and transaction data. Once a block is added to the chain, it cannot be altered or deleted without invalidating subsequent blocks, making the system resistant to tampering and ensuring the integrity of the data.

3.What is Ethereum Block-Chain ?
Ethereum is an open-source, decentralized blockchain platform. which is a decentralized network that enables developers to build and run decentralized applications, smart contracts. One of the most significant differences is that Ethereum supports the execution of smart contracts.

4.What is Smart-Contract ?
A smart contract is a computer program that automatically executes the terms of a contract when certain pre-defined conditions are met. These programs are designed to run on a blockchain, which is a decentralized and distributed digital ledger technology that allows for secure and transparent transactions without the need for intermediaries.

5.What is Hash Value ?
A hash value, also known as a hash code, is a numerical representation of data that is generated by a hash function. Hash functions are mathematical algorithms that take in data of arbitrary size and produce a fixed-size output, typically a string of numbers and letters.

6.What Secure Hash Algorithm(SHA-256) ?
SHA-256 is a cryptographic hash function that generates a fixed-size output (256 bits) from an input message of arbitrary length.And is considered to be one of the most secure hash functions available.
